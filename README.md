# DVMOTORS - Sistema de Gestión de Taller Mecánico

Sistema web completo para la gestión de citas y mecánicos de un taller automotriz.

## Características

- **Autenticación de usuarios** (Clientes y Administrador)
- **Sistema de citas** para clientes
- **Panel de administración** completo
- **Gestión de mecánicos** con reseñas y calificaciones
- **Diseño responsivo** para móviles y escritorio
- **Modo oscuro/claro**
- **Perfiles de usuario** personalizables

## Instalación en GitHub Pages

1. Crea un repositorio en GitHub llamado `dvmotors`
2. Sube todos los archivos a la raíz del repositorio
3. Ve a Settings → Pages
4. Selecciona la rama `main` como source
5. Tu sitio estará disponible en: `https://tuusuario.github.io/dvmotors`
